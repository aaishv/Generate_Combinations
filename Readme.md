# Generate Combinations

## Steps to run the project:
```bash

git clone https://github.com/aaishv/Generate_Combinations.git 

cd backend

mvn spring-boot:run

http://localhost:8080/api/1234567 (On Browser)

cd frontend

npm install

ng serve

http://localhost:4200 (On Browser)
```
